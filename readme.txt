Admin pages list Image preview 

== Description ==

Image preview for every page in wp admin pages list, no need to click 'View' anymore.
You can resize the frame view to feet with other plugins columns.

== Installation ==

1. From your WordPress Dashboard go to 'Plugins' > 'Add new' and search for 'Page List Preview'.
2. Click 'Install Now' under the title of the Call Now Button plugin
3. Click 'activate'
4. You're done, now you well see the preview in the pages list, until you deactivate this plugin.

Or:

1. Click download on this page and download the "page_list_preview'.zip" file
2. From your WordPress Dashboard go to 'Plugins' > 'Upload plugin' and select the "page_list_preview.zip" file
3. Click 'install'
4. Click 'activate'
5. Activate the plugin through the 'Plugins' menu in WordPress
4. You're done, now you well see the preview in the pages list, until you deactivate this plugin.
